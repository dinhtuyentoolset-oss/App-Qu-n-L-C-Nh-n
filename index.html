Đang kiểm tra phiên bản…

Mỗi file HTML (gốc / bản sao) có mã version riêng. Chỉ khi bạn chốt thì Telegram mới gắn file đó — mở file cũ sẽ không cướp bot.

📦 Google Sheet (backend cố định)

App chỉ dùng một Sheet gắn với Apps Script. Link Web App cấu hình một lần trong config.js (GS_WEBAPP_URL). Sau này sửa code: Deploy → Manage deployments → Edit → New version — giữ nguyên link, dữ liệu user vẫn trong tab Data_[username].

🤖 Telegram Bot

Số ID (không phải @username). Chat cá nhân: nhắn bot /start trước → bấm «Lấy Chat ID». Nhóm: thêm bot vào nhóm, ID thường bắt đầu -100.

🔔 Nhắc nhở

Cài lịch bằng 🔔 trên từng mục (tự lưu lên Sheet). Đóng app vẫn nhận Telegram qua Apps Script. Đăng nhập nhiều thiết bị: dữ liệu tự đồng bộ ~45 giây khi mở app.
